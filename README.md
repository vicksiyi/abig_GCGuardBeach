 ## :black_nib: 项目简介:pencil2:
 为用户提供一个具有趣味性和公益性的平台。用户可以在小程序上传自己发现的海滩垃圾照
片。志愿者可以通过这款小程序聚集起来，进而更好地为重建蓝色海洋作一份贡献
	
## :whale: 1、技术栈
- 前端：微信小程序原生代码、IviewUI组件库
- 后端：Node、Express、云开发
- 通信：WebSocket
- 数据库：MongoDb

## :frog: 2、系统模块
- 登录授权模块
- 新闻列表模块
- 海洋知识模块
- 发现圈模块
- 商品兑换模块
- 我的模块
